o que é

O flow.planner é um sistema pessoal de frequência acadêmica e agenda semanal. Ele foi criado para ajudar estudantes a acompanharem suas presenças, tarefas e tópicos de estudo de forma simples e eficiente, diretamente no navegador.

recursos principais

  • Agenda semanal interativa com visualização por semestre
  • Controle de frequência por disciplina com cálculo automático de percentual
  • Criação de eventos, lembretes, provas e entregas
  • Anotações de tarefas e tópicos de estudo
  • Sincronização com a nuvem via Supabase (opcional)
  • Modo escuro e suporte a PWA (instalável no celular)
  • Funcionamento offline com sincronização quando a conexão retornar

filosofia

O flow.planner foi desenvolvido com foco em leveza e praticidade. Não exige cadastro obrigatório — você pode usá-lo imediatamente como convidado, com todos os dados salvos localmente no seu dispositivo. A sincronização com a nuvem é opcional e ativada apenas quando você escolhe criar uma conta.

tecnologia

  • HTML, CSS e JavaScript puros — sem frameworks pesados
  • PWA com service worker para uso offline
  • Supabase como backend opcional de sincronização
  • Armazenamento local via IndexedDB / localStorage
