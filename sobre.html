o que é

O flow.planner é um sistema pessoal de frequência acadêmica e agenda semanal desenvolvido para estudantes universitários. Ele foi criado para facilitar o acompanhamento de presenças, organização de tarefas e controle de tópicos de estudo, tudo de forma simples, rápida e diretamente no navegador — sem instalações ou configurações complexas.

recursos principais

  • Agenda semanal interativa com visualização por semana e navegação por teclado (← →)
  • Controle de frequência por disciplina com cálculo automático de percentual e alerta de limite de faltas
  • Criação de eventos personalizados: lembretes, provas, entregas e outros tipos
  • Arrastar e soltar eventos no calendário para remarcação rápida (desktop e mobile)
  • Anotações organizadas em listas de tarefas e tópicos de estudo
  • Sincronização com a nuvem via Supabase (opcional, com conta gratuita)
  • Modo escuro e PWA: instalável no celular como aplicativo nativo
  • Funcionamento offline com sincronização automática ao reconectar
  • Exportação e importação de dados em formato JSON para backup local

filosofia

O flow.planner foi desenvolvido com foco em leveza, privacidade e praticidade. Não exige cadastro obrigatório — você pode começar a usar imediatamente como convidado, com todos os dados salvos localmente no seu próprio dispositivo.

A sincronização com a nuvem é totalmente opcional e ativada apenas quando você opta por criar uma conta. Nenhum dado é coletado sem o seu consentimento, e nenhuma telemetria é enviada para servidores externos no modo convidado.

tecnologia

  • HTML, CSS e JavaScript puros — sem frameworks pesados, carregamento instantâneo
  • Progressive Web App (PWA) com service worker para uso offline completo
  • Supabase como backend opcional de autenticação e sincronização em nuvem
  • IndexedDB / localStorage para armazenamento local persistente nos dispositivos
  • Design responsivo otimizado para desktop, tablet e smartphone

versão e atualizações

O flow.planner está em constante evolução. Melhorias, correções e novos recursos são lançados periodicamente. Por ser um PWA, as atualizações são aplicadas automaticamente quando você acessa o sistema com conexão à internet.
