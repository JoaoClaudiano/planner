2. descrição do serviço

O flow.planner é uma aplicação web de uso pessoal para controle de frequência acadêmica, agenda semanal e anotações. O serviço pode ser utilizado sem cadastro (modo convidado, com dados armazenados exclusivamente no dispositivo do usuário) ou com conta opcional para sincronização em nuvem via Supabase.

4. dados e privacidade

Seus dados acadêmicos são armazenados localmente no seu dispositivo quando você usa o modo convidado. Quando você opta pela sincronização em nuvem, os dados são armazenados de forma segura via Supabase, com criptografia em trânsito (HTTPS/TLS) e isolamento por usuário (Row Level Security). Consulte nossa Política de Privacidade para mais detalhes.
