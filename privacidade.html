flow.studio

política de privacidade

Última atualização: março de 2025

1. introdução

O flow.studio respeita a sua privacidade. Esta política descreve quais informações coletamos, como as utilizamos e como as protegemos.

2. dados coletados

  • Modo convidado: nenhum dado é enviado a servidores externos. Todas as informações (agenda, frequências, anotações) são armazenadas exclusivamente no seu dispositivo via localStorage/IndexedDB.
  • Modo com conta: seu endereço de e-mail é coletado para autenticação. Os dados acadêmicos são sincronizados e armazenados no Supabase de forma segura.
  • Dados de uso: não coletamos dados de navegação, telemetria ou rastreamento de comportamento.

3. uso das informações

  • Autenticação e identificação do usuário (somente quando há conta)
  • Sincronização dos seus dados entre dispositivos (somente quando há conta)
  • Os dados nunca são vendidos, compartilhados ou utilizados para fins publicitários

4. armazenamento e segurança

Os dados em nuvem são armazenados via Supabase, com transmissão criptografada (HTTPS/TLS). Cada usuário só tem acesso aos próprios dados, garantidos por políticas de segurança em nível de linha (Row Level Security).

5. cookies e armazenamento local

O flow.studio utiliza localStorage e IndexedDB do navegador para salvar preferências (como modo escuro) e dados do usuário localmente. Não utilizamos cookies de rastreamento de terceiros.

6. exclusão de dados

Você pode apagar todos os dados locais limpando o armazenamento do navegador para este site. Para excluir dados em nuvem, entre em contato com nossa equipe.

7. alterações nesta política

Podemos atualizar esta política periodicamente. Recomendamos revisá-la regularmente. O uso continuado do serviço após alterações implica aceitação da política revisada.

8. contato

Para questões sobre privacidade, envie uma mensagem pela nossa página de contato.
