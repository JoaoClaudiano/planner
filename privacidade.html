flow.planner

política de privacidade

Última atualização: março de 2025 · Comprometemo-nos com a proteção dos seus dados.

1. introdução

O flow.planner foi construído com privacidade por princípio. Esta política descreve quais informações coletamos, como as utilizamos, como as protegemos e quais são os seus direitos como usuário.

2. dados coletados

  • Modo convidado: nenhum dado é enviado a servidores externos. Todas as informações (agenda, frequências, anotações) são armazenadas exclusivamente no seu dispositivo via localStorage/IndexedDB.
  • Modo com conta: seu endereço de e-mail é coletado para autenticação. Os dados acadêmicos são sincronizados e armazenados no Supabase de forma segura e criptografada.
  • Dados de uso: não coletamos dados de navegação, telemetria ou rastreamento de comportamento em nenhum dos modos.
  • Mensagens de contato: se você enviar uma mensagem pelo formulário de contato, os dados fornecidos (nome, e-mail e mensagem) serão utilizados exclusivamente para responder à sua solicitação.

3. uso das informações

  • Autenticação e identificação do usuário (somente quando há conta)
  • Sincronização dos seus dados entre dispositivos (somente quando há conta)
  • Os dados nunca são vendidos, compartilhados com terceiros ou utilizados para fins publicitários
  • Não realizamos perfilamento ou análise comportamental dos usuários

4. armazenamento e segurança

Os dados em nuvem são armazenados via Supabase, com transmissão criptografada (HTTPS/TLS). Cada usuário só tem acesso aos próprios dados, garantidos por políticas de segurança em nível de linha (Row Level Security — RLS). O Supabase está em conformidade com padrões de segurança internacionais.

5. cookies e armazenamento local

O flow.planner utiliza localStorage e IndexedDB do navegador para salvar preferências (como modo escuro) e dados do usuário localmente. Não utilizamos cookies de rastreamento de terceiros nem ferramentas de analytics.

6. seus direitos

  • Acesso: você pode exportar todos os seus dados a qualquer momento pelo botão "exportar" no aplicativo.
  • Exclusão local: limpe o armazenamento do navegador para este site para remover todos os dados locais.
  • Exclusão em nuvem: entre em contato para solicitar a exclusão completa dos seus dados em nuvem.
  • Portabilidade: seus dados podem ser exportados em formato JSON a qualquer momento.

7. alterações nesta política

Podemos atualizar esta política periodicamente. Recomendamos revisá-la regularmente. O uso continuado do serviço após alterações implica aceitação da política revisada. Mudanças significativas serão destacadas na data de atualização.

8. contato

Para questões sobre privacidade ou exercício dos seus direitos, envie uma mensagem pela nossa página de contato.
